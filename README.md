OpenWrt Terminal Pro v2.4.4
Терминал и файловый менеджер для Xiaomi AX3000T (OpenWrt) с единым SSH/SFTP подключением.

Возможности
SSH-терминал с цветным выводом и историей команд

Двухпанельный файловый менеджер (локальный ПК ↔ роутер)

Загрузка и скачивание файлов по SFTP с прогресс-баром

Панель быстрых команд (статус системы, WiFi, firewall, zapret и др.)

Настраиваемые макросы с возможностью импорта/экспорта

Современный тёмный интерфейс без стандартных рамок Windows

Автоматическая установка SFTP на роутер

Установка
Скачай OpenWrt Terminal Pro v2.4.4.exe из раздела Releases

Запусти — программа сама установит paramiko, если нужно

Подключись к роутеру (по умолчанию 192.168.1.1:22, логин root)

Системные требования
Windows 10/11

Роутер Xiaomi AX3000T с OpenWrt (подойдут и другие роутеры с SSH)

Установленный openssh-sftp-server на роутере (ставится кнопкой "SFTP" в программе)
